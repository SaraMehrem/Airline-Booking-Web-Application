This project is a full-stack web application that allows users to browse available flights, create accounts, and manage their bookings. It includes features such as user registration and login, booking flights, updating or canceling reservations, and viewing flight information. The system is designed with a responsive user interface and incorporates SEO techniques to improve visibility. It was developed using HTML, CSS, JavaScript for the front-end, PHP for the back-end, and MySQL for database management.
